Приложение предназначено для парсинга словаря, его селекции и визуализации. Инструмент для проверки гипотезы морфемной классификативности русского языка.

Word.cs
Класс описывает слово: набор приставок, корень, набор суффиксов, окончание, само слово, транскрипция.
Конструкторы, что классифицирует, ToString() и ToStringRoot().

Record.cs
Окружение (Pair) и соответствующие ему корни.

Pair.cs
Класс описывает 1 аффиксальное окружение (лист приставок и лист соответствующих им суффиксов), ToString(), FromString() и конструкторы.

FullEnvironment.cs
Класс описывает все аффиксалльные окружения 1 корня. Реализовано добавление окружения, ToString() с повторениями и ToString() для сета: окружение и количество раз, которые оно встречается без повторений.

AllAffixEnvironments.cs
Предназначен для нахождения всех аффиксальных окружений*, встречающихся в словаре с возможностью их вывода в текстовый файл и в Excel-таблицу.

ExtractRoot.cs
Предназначен для нахождения всех корней, встречающихся в словаре с возможностью их вывода в текстовый файл.

Parser.cs
Наиболее значимый класс. Общий парсер. На вход получает словарь. На выход 2 множества с/без подходящими корнями (перечислены все слова, т.е. корни повторяются).
Парсинг строки. Проверки состава слова (наличие приставок, суффиксов, проклитики, энклитики).

StatisticRoot.cs
Парсер для аффиксов. 
Нахождение всех (полного) аффиксальных окружений для каждого корня из поданного на вход списка*/

Streamer.cs
Вспомогательный класс. Читать из файла в IEnumerable, List и Печать 
CreateListOfRootSets(), HasRootInSet(), CreateRootSet()
RemoveInvalidSymbols()
генерация имен файлов с множествами и их пересечениями

Set.cs
Возвращает множество, являющиеся пересечением множеств корней (полученных в результате обработки Parserа)

Состоит из нескольких Форм:
0. Базавая UniForm. Надкласс для дочерних форм. Реализует визуализацию деревьев на форме.
1. Основная Form1. Необходима для создания множеств по введённому аффиксальному окружению* или корню. Выпадающие списки заполняются автоматически, есть проверки ввода. На выходе адреса файлов - сочетающихся и несочетающихся корней/аффиксальных окружений.
2. Дочерние формы PrefForm, SufForm. Необходимы для работы с приставочными и суффиксальными** окружениями. Строится trie-дерево с возможностью раскрытия всех узлов без динамической подгрузки. Можно построить множество слов(корней) по выбранному поддереву. На выходе адрес файла - сочетающиеся с аффиксальным окружением отсортированные слова. 
3. Вспомогательная форма FormTimer. Необходим для отсчёта времени формирования дерева в дочерних формах. В последствии, был исключен.


* Аффиксальное окружение - последовательный набор приставок и суффиксов слова.
**Приставочное окружение - последовательный набор приставок слова.
**Суффиксальное окружение - последовательный набор суффиксов слова.
